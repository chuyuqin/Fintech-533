Duke FinTech 533 • Breakout Strategy Project

Single-stock breakout research report

I tested a fixed long-only breakout rule set on a liquid large-cap stock universe, used daily bars to define the setup, used hourly bars for the first clean breakout entry, and evaluated each candidate through rolling walk-forward out-of-sample windows. The strongest stock in this run was MU, which is the focus asset for the trade ledger and performance analysis below.

Strategy logic

The strategy now follows one fixed rule set instead of a parameter sweep. Daily bars define the setup: the breakout level is the highest high of the prior 20 trading days, yesterday's close must be above the 50-day EMA, and yesterday's volume must be at least 80% of its 20-day average. The intended execution logic is to buy the first clean breakout after that setup, risk 1% of equity, place the stop at 1.2 ATR, set the target at 2.4 ATR, and exit after 5 trading days if neither threshold is hit.

Asset selection

I screened 8 liquid large-cap stocks (MU, MSFT, META, AVGO, AAPL, LLY, NVDA, AMZN) using the same fixed breakout rule set and the same rolling walk-forward framework with about one year of training data and the next quarter as out-of-sample test data. Quantitatively, MU ranked best after combining average out-of-sample Sharpe ratio (1.61), total out-of-sample return (5.90%), drawdown control, and trade count (10). Qualitatively, I preferred stocks with deep liquidity, sustained institutional participation, and a narrative that can support trend persistence without overwhelming the chart with one-off event risk. For MU, that qualitative case was: Semiconductor cyclicality often creates tradable expansions, but earnings and guidance risk are meaningful.

Breakout definition and assumptions

The breakout detection function is implemented in Python in breakout_report_generator.py. The parameters are explicit and easy to locate at the top of the script. The revised research design focuses on a single-stock universe instead of ETFs and combines quantitative evidence with qualitative judgment when picking the final candidate. Quantitatively, stocks are ranked by out-of-sample Sharpe ratio, out-of-sample return, drawdown, and trade count under the same fixed rule set. Qualitatively, I favor names with deep liquidity, durable institutional participation, and repeated breakout behavior while penalizing names whose gaps or event risk can overwhelm a clean trend-following setup. The backtest uses IBKR data through shinybroker, assumes 2 bps slippage per side, $0.005 per share commissions with a $1 minimum, and sizes positions so that the initial stop risks about 1% of current equity. The annual risk-free rate in the Sharpe ratio calculation is 2%.
